# LeyLineBook v1.5.0

本版本新增版本检查和自动更新能力，并继续保留 1.4.0 的游戏日刷新与角色背景调节。

## 主要更新

- 新增启动时静默检查 GitHub Releases 是否有新版本。
- 新增设置页“版本与更新”，可手动检查更新。
- 发现新版本时会在页面顶部提示，可选择立即更新或稍后处理。
- 打包后的 EXE 支持自动下载新版并重启替换。
- 自动更新优先下载匹配当前版本号的 `LeyLineBook-v版本号-Windows-x64.exe`。
- 今日任务、活动默认日期和备份日期继续按凌晨 4 点后的游戏日计算。
- 角色主题支持图片比例和图片位置调节。

## Windows 使用方法

1. 下载 `LeyLineBook-v1.5.0-Windows-x64.exe`。
2. 双击 EXE 启动，首次启动可能会稍慢。
3. 数据库和日志会保存在 `%APPDATA%\LeyLineBook`。
4. 后续版本可在“设置与备份”的“版本与更新”中检查。

导出的 JSON 备份不包含账号凭据，导入后需要重新填写。请勿将真实数据库、日志或账号信息上传到 GitHub。发行包不包含《原神》官方角色图片，也不包含真实号主数据。

# LeyLineBook v2.0.0

本版本优化周期任务的时间提示，让今日任务界面更简洁。

## 主要更新

- “壶”完成后显示下一次收取日期，未完成时不再显示容易混淆的到期提示。
- 每周任务完成后显示下周一 04:00 的刷新时间。
- 质变仪与探索派遣冷却期间继续集中显示在冷却区域。
- 每周任务在当前周期内保持完成状态，到下周一 04:00 自动刷新。
- 修复周期任务在今日任务页面中的归类兼容问题。

## Windows 使用方法

1. 下载 `LeyLineBook-v2.0.0-Windows-x64.exe`。
2. 双击 EXE 启动，首次启动可能会稍慢。
3. 数据库和日志保存在 `%APPDATA%\LeyLineBook`。
4. 后续版本可在“设置与备份”的“版本与更新”中检查。

导出的 JSON 备份不包含账号凭据，导入后需要重新填写。请勿将真实数据库、日志或账号信息上传到 GitHub。

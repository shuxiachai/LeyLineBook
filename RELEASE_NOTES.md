# LeyLineBook v1.7.0

本版本继续完善自动更新体验，并修正周本备注区域的显示方式。

## 主要更新

- 应用版本更新为 v1.7.0。
- 自动更新下载后会按新版本号生成 `LeyLineBook-v版本号-Windows-x64.exe`。
- 自动更新完成后会启动新版 EXE。
- 修正任务备注弹窗中周本区域的隐藏方式，避免布局状态不一致。
- 继续保留 v1.6.0 的体力周本快捷备注功能。

## Windows 使用方法

1. 下载 `LeyLineBook-v1.7.0-Windows-x64.exe`。
2. 双击 EXE 启动，首次启动可能会稍慢。
3. 数据库和日志会保存在 `%APPDATA%\LeyLineBook`。
4. 后续版本可在“设置与备份”的“版本与更新”中检查。

导出的 JSON 备份不包含账号凭据，导入后需要重新填写。请勿将真实数据库、日志或账号信息上传到 GitHub。发行包不包含《原神》官方角色图片，也不包含真实号主数据。

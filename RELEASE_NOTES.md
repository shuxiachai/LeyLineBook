# LeyLineBook v1.4.0

本版本继续优化角色主题和页面导航，让本地记录界面更容易调整成自己喜欢的样子。

## 主要更新

- 角色主题新增图片比例调节，可放大或缩小背景角色图。
- 角色主题新增图片位置调节，可在顶部、居中、底部之间微调构图。
- 今日任务、活动默认日期和备份日期统一按凌晨 4 点后的游戏日计算。
- 左侧导航文案调整，“活动任务”改为“限时活动”。
- 左侧导航顺序调整，剧情任务入口前移。
- 新增号主示例改为“例如：树下”。
- 保持单文件 EXE 数据目录为 `%APPDATA%\LeyLineBook`，避免程序目录产生真实数据库。
- 继续保留 JSON 备份导入、清空数据、加密凭据和本地角色主题功能。

## Windows 使用方法

1. 下载 `LeyLineBook-v1.4.0-Windows-x64.exe`。
2. 双击 EXE 启动，首次启动可能会稍慢。
3. 数据库和日志会保存在 `%APPDATA%\LeyLineBook`。
4. 导出的 JSON 备份不包含账号凭据，导入后需要重新填写。

请勿将真实数据库、日志或账号信息上传到 GitHub。发行包不包含《原神》官方角色图片，也不包含真实号主数据。

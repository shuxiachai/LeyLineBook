# LeyLineBook v1.2.0

本版本新增本地账号凭据管理，并改善单文件 Windows 程序的后台关闭体验。

## 主要功能

- 今日任务、每日进度、总进度与历史记录
- 号主、任务、活动和剧情任务管理
- 深境螺旋、幻想真境剧诗与危战周期计算
- 质变仪精确到小时的冷却记录
- 黑、白、绿三种纯色主题
- 本地角色主题：自动提取图片主色与明暗，生成全屏背景和磨砂界面
- 本地 SQLite 数据库、JSON 备份与虚构演示数据库
- 账号名称、密码和凭据备注使用 Windows DPAPI 加密保存
- 关闭所有地脉簿页面约 90 秒后，后台程序自动退出

## Windows 使用方法

1. 下载 `LeyLineBook-v1.2.0-Windows-x64.exe`。
2. 将 EXE 放在具有写入权限的文件夹中。
3. 双击 EXE，程序会自动打开浏览器；首次启动可能稍慢。
4. 数据库和日志会保存在 EXE 所在目录。
5. 在“设置与备份”中可以选择本地角色图片、导出备份或关闭程序。

账号凭据仅能由保存时的 Windows 用户账户解密，导出的 JSON 备份不包含凭据。请勿将真实数据库或日志上传到 GitHub。

发行包不包含《原神》官方角色图片，也不包含真实号主数据。

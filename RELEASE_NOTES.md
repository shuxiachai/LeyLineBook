# LeyLineBook v1.8.0

本版本继续修正任务备注交互，让周本备注和专项任务标签更符合实际使用习惯。

## 主要更新

- 应用版本更新为 v1.8.0。
- 切换“好感队”或“委托”时，会保留体力任务里已经选择的周本备注。
- 深境螺旋、幻想真境剧诗和危战改为直接添加或移除，不再弹出备注设置窗口。
- 修正彻底删除号主弹窗的确认按钮文字。
- 新增测试覆盖周本备注保留逻辑。

## Windows 使用方法

1. 下载 `LeyLineBook-v1.8.0-Windows-x64.exe`。
2. 双击 EXE 启动，首次启动可能会稍慢。
3. 数据库和日志会保存在 `%APPDATA%\LeyLineBook`。
4. 后续版本可在“设置与备份”的“版本与更新”中检查。

导出的 JSON 备份不包含账号凭据，导入后需要重新填写。请勿将真实数据库、日志或账号信息上传到 GitHub。发行包不包含《原神》官方角色图片，也不包含真实号主数据。

# LeyLineBook v1.3.0

本版本新增 JSON 备份导入、数据库清空和四种纯色主题，并将单文件版的数据统一保存到 Windows 应用数据目录。

## 主要功能

- 今日任务、每日进度、总进度与历史记录
- 号主、任务、活动和剧情任务管理
- 深境螺旋、幻想真境剧诗与危战周期计算
- 质变仪精确到小时的冷却记录
- 黑、白、绿、蓝、紫、玫瑰、琥珀七种纯色主题
- 本地角色主题：自动提取图片主色与明暗，生成全屏背景和磨砂界面
- 本地 SQLite 数据库、JSON 备份与虚构演示数据库
- 账号名称、密码和凭据备注使用 Windows DPAPI 加密保存
- 关闭所有地脉簿页面约 90 秒后，后台程序自动退出
- JSON 备份支持导出和导入，导入前提供二次确认
- 支持二次确认后清空全部任务数据
- 单文件版数据库与日志保存到 `%APPDATA%\LeyLineBook`
- 首次升级时自动迁移 EXE 旁的旧数据库

## Windows 使用方法

1. 下载 `LeyLineBook-v1.3.0-Windows-x64.exe`。
2. 将 EXE 放在具有写入权限的文件夹中。
3. 双击 EXE，程序会自动打开浏览器；首次启动可能稍慢。
4. 数据库和日志会保存在 `%APPDATA%\LeyLineBook`。
5. 在“设置与备份”中可以选择本地角色图片、导入或导出备份、清空数据或关闭程序。

账号凭据仅能由保存时的 Windows 用户账户解密，导出的 JSON 备份不包含凭据。请勿将真实数据库或日志上传到 GitHub。

发行包不包含《原神》官方角色图片，也不包含真实号主数据。

# LeyLineBook v1.9.0

本版本新增探索派遣记录，并整理今日任务与旧任务类型的显示逻辑。

## 主要更新

- 新增“探索派遣”任务，可选择 15 小时或 20 小时派遣周期。
- 完成派遣时可记录实际收取时间，并实时显示距离下次收取的剩余时间。
- 质变仪与探索派遣在冷却期间统一显示在冷却区域。
- 移除“专项记录”和“随时记录”入口；旧版随时记录任务会自动转换为一次性任务，不会丢失。
- 周任务完成后会在本周内保持勾选状态，到下周一 04:00 自动刷新。

## Windows 使用方法

1. 下载 `LeyLineBook-v1.9.0-Windows-x64.exe`。
2. 双击 EXE 启动，首次启动可能会稍慢。
3. 数据库和日志保存在 `%APPDATA%\LeyLineBook`。
4. 后续版本可在“设置与备份”的“版本与更新”中检查。

导出的 JSON 备份不包含账号凭据，导入后需要重新填写。请勿将真实数据库、日志或账号信息上传到 GitHub。

# LeyLineBook v1.6.0

本版本优化每日任务备注体验，重点改善体力任务的周本记录方式。

## 主要更新

- 体力任务备注中新增“周本”快捷标签。
- 今日任务页面会把普通备注和周本备注分开展示。
- 狗粮和爱可菲料理改为直接添加或移除，不再弹出备注设置窗口。
- 号主与任务页面的分组标题字号加大，更容易扫视。
- 应用版本更新为 v1.6.0，继续保留自动检查更新功能。

## Windows 使用方法

1. 下载 `LeyLineBook-v1.6.0-Windows-x64.exe`。
2. 双击 EXE 启动，首次启动可能会稍慢。
3. 数据库和日志会保存在 `%APPDATA%\LeyLineBook`。
4. 后续版本可在“设置与备份”的“版本与更新”中检查。

导出的 JSON 备份不包含账号凭据，导入后需要重新填写。请勿将真实数据库、日志或账号信息上传到 GitHub。发行包不包含《原神》官方角色图片，也不包含真实号主数据。

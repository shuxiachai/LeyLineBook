# LeyLineBook v1.1.1

Windows 打包修订版本：发行程序改为单个 EXE，不再依赖 `_internal` 文件夹。

同时修复部分 Windows 电脑拒绝使用默认端口时无法启动的问题；程序现在会自动选择其他可用的本地端口。

## 主要功能

- 今日任务、每日进度、总进度与历史记录
- 号主、任务、活动和剧情任务管理
- 深境螺旋、幻想真境剧诗与危战周期计算
- 质变仪精确到小时的冷却记录
- 黑、白、绿三种纯色主题
- 本地角色主题：自动提取图片主色与明暗，生成全屏背景和磨砂界面
- 本地 SQLite 数据库、JSON 备份与虚构演示数据库

## Windows 使用方法

1. 下载 `LeyLineBook-v1.1.1-Windows-x64.exe`。
2. 将 EXE 放在具有写入权限的文件夹中。
3. 双击 EXE，程序会自动打开浏览器；首次启动可能稍慢。
4. 数据库和日志会保存在 EXE 所在目录。
5. 在“设置与备份”中可以选择本地角色图片、导出备份或关闭程序。

发行包不包含《原神》官方角色图片，也不包含真实号主数据。

# LeyLineBook / 地脉簿

这是一个纯本地记录工具，不会连接或操作游戏。

## 启动

双击 `启动地脉簿.vbs`，程序会在后台运行并自动打开浏览器，不会显示黑色命令窗口。再次双击会正常重启后台并打开最新版页面。

GitHub Release 提供单文件版 `LeyLineBook.exe`，无需安装 Python，也不依赖 `_internal` 文件夹。首次启动时需要解压内置运行资源，因此可能比后续启动稍慢。打包版可在“设置与备份”中检查 GitHub Releases 新版本。

关闭所有地脉簿页面后，后台程序会在约 90 秒内自动退出；也可以进入“设置与备份”，点击“关闭程序”立即停止。

单文件 EXE 的数据保存在 `%APPDATA%\LeyLineBook\task_records.db`，日志保存在同一目录。首次升级时，如果 EXE 旁存在旧数据库且新目录尚无数据库，程序会自动复制过去。删除数据库后，程序只会创建空数据库，不会读取 Excel 或生成内置名单。本程序只支持自身 JSON 备份导入，不提供 Excel 导入功能。

## 主要功能

- 今日任务：一键完成、撤销、全部完成、切换日期补记
- 周期任务：深境螺旋、幻想真境剧诗、危战会按各自刷新周期显示
- 剧情任务：按号主记录魔神任务、传说任务和世界任务，可标记额外奖励期限
- 号主管理：新增、修改、停用号主，按住右侧拖动把手可调整顺序
- 凭据管理：可选保存账号名称、密码和凭据备注，使用当前 Windows 用户的 DPAPI 加密
- 任务管理：每日、间隔周期、一次性任务；体力任务可用快捷标签记录周本
- 历史记录：按日期范围查询
- 数据备份：设置页可导出或导入 JSON 文件，也可在二次确认后清空数据库
- 版本更新：启动时静默检查新版本，设置页可手动检查并更新打包版 EXE
- 界面主题：黑、白、绿、蓝、紫、玫瑰、琥珀七种主色调，选择后自动记忆
- 角色主题：从本机选择图片后自动提取主色与明暗，启用全屏背景和磨砂界面；点击任一纯色主题即可恢复纯色，图片只保存在当前浏览器

## 固定刷新规则

- 幻想真境剧诗：每月 1 日刷新
- 深境螺旋（深渊）：每月 16 日刷新
- 危战：版本更新后第 8 天（第二周星期三）10:00 开放，版本最后一周星期二 03:59 关闭

危战以官方 Luna VII 更新日 `2026-05-20` 为锚点，按原神常规42天大版本周期自动滚动。危战在版本第8天10:00开放，并在下一版本更新前一天03:59截止。只有官方发生延期或改期时，才需要在“设置与备份”中校准版本日期。

## 每日任务分类

- 体力：备注可标注“好感队”或“委托”
- 狗粮：作为独立的每日任务记录
- 捡材料：暂时保持原任务名称，不归入体力或狗粮

“号主与任务”页面分为两组标签：

- 每日：体力、狗粮、质变仪（168小时）、壶（3天）、爱可菲料理（每周一 04:00 刷新）、探索派遣（15或20小时），体力备注包含好感队、委托、周本和自定义备注
- 深渊：深境螺旋、剧诗、危战，支持独立的自定义备注

点击任务标签即可启用或停用；捡材料已从任务标签中移除，旧历史记录仍保留。

“爱可菲料理”每周任意时间完成一次即可，完成状态保持到下周一 04:00，并计入每日进度。

## 剧情任务

- 魔神任务：可选额外奖励，截止到当前版本末周二 15:00
- 传说任务：可选额外奖励，截止到当前半版本末周二 15:00
- 世界任务：默认无额外奖励期限

奖励期限结束后任务不会消失，仍会保留在待完成列表中。
剧情任务的号主既可以从现有名单选择，也可以直接输入临时名称；任务名称留空时会自动使用任务类型名称。

## 数据安全

所有数据都在本机。账号凭据使用 Windows DPAPI 加密，仅能由保存凭据时的 Windows 用户账户解密；导出的 JSON 备份不包含凭据，导入后需要重新填写。请勿在普通备注中填写密码或验证码，也不要把真实数据库上传到 GitHub。建议定期导出 JSON 备份；数据库中的加密凭据换电脑或更换 Windows 用户后无法解密。

## 演示数据

仓库中的 `demo/task_records.demo.db` 只包含虚构数据，可用于截图、录屏和功能展示。使用方法见 [`demo/README.md`](demo/README.md)。请勿用演示库覆盖自己的真实数据库。

## 角色主题与素材说明

角色主题支持用户从本机选择图片，图片仅保存在当前浏览器的本地存储中，不会写入任务数据库、备份文件或公开发行包。本仓库不包含《原神》官方角色立绘、音乐、语音或其他游戏素材。

## 开发与打包

项目运行时只使用 Python 标准库，开发环境建议使用 Python 3.11 或更高版本。

```powershell
python -m pip install -r requirements-dev.txt
python app.py
python -m unittest discover -s tests -v
python -m PyInstaller --clean -y LeyLineBook.spec
```

数据库、日志、缓存和打包结果均不应提交到源码仓库，具体规则见 `.gitignore`。

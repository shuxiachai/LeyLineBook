快速开始

地脉簿是一款本地运行的原神多账号任务追踪工具。双击 EXE 后程序在后台启动，自动在浏览器中打开界面。所有数据仅保存在本机，不上传至任何服务器。

基本使用流程

  1. 在「号主与任务」页新增号主（每个游戏账号对应一个号主）
  2. 点击号主卡片中的任务按钮，开启需要追踪的任务
  3. 每天在「今日任务」页查看待办并逐一点击完成

程序关闭后任务记录仍会保留。下次打开时，已完成的每日任务会自动在次日 04:00 重置。

号主与任务

「号主」对应一个游戏账号，所有任务都归属于某个号主，在今日任务页按号主分组显示。

新增号主

点击页面右上角 新增号主，填写以下信息：

  • 名称：在任务列表中显示的标识，建议填写账号昵称或编号
  • 联系人/分组（可选）：填写账号持有人或分组名称，便于多号管理
  • 截止日期（必填）：代打服务的到期日期。若非代打可填写一个远期日期，临近到期时号主名称旁会出现提醒标记
  • 备注（可选）：自由填写的额外说明
  • 账号凭据（可选）：可在新增时一并填写账号密码，也可后续通过 密 按钮添加

任务配置

每个号主卡片分三个区域：

  • 每日：体力、狗粮、质变仪、壶、爱可菲料理、探索派遣
  • 活动：在「限时活动」页添加的活动任务（可按号主单独开关）
  • 深渊：深境螺旋、幻想真境剧诗、危战

按钮显示 ✓ 时表示已为该号主启用，点击可切换。部分任务点击后会弹出备注设置对话框：

  • 探索派遣：选择冷却时长（20小时 或 15小时），按号主角色阵容决定
  • 深境螺旋 / 幻想真境剧诗：可记录挑战层数（可选，不影响追踪逻辑）

安全凭据存储

每个号主卡片右上角有 密 按钮，可安全存储游戏账号和密码。采用 Windows DPAPI 加密，只有本机才能解密，账号密码不会出现在备份导出文件中。

请勿将密码写在备注字段中。迁移到新设备后，需重新填写凭据。

调整排序

长按并拖动号主卡片右侧的 ↕ 图标，可调整号主在今日任务页的显示顺序。

停用与删除

  • ✕ 停用：号主及其任务从列表隐藏，历史记录保留，可随时通过 ↺ 重新启用
  • ␡ 彻底删除：号主从界面中永久移除，任务不再显示；历史完成记录仍保留在数据库中，可在「历史记录」页继续查询

彻底删除操作不可撤销，执行前请先确认数据已备份或不再需要。

今日任务

游戏日与重置时间

原神以每天凌晨 04:00 为一天的分界点。软件与游戏保持一致——当前时间在 00:00–03:59 时，显示的仍是前一天任务周期内的内容，完成记录也会归入前一天。

顶部汇总卡片

  • 总进度：所有今日到期任务中已完成的数量及进度条
  • 每日进度：体力/狗粮/质变仪/壶/爱可菲料理/探索派遣 的专项进度，这类任务按各自冷却或周期出现，不一定每天都需要完成
  • 待完成：当前还未完成的任务数量

完成与撤销

点击任务卡片即可标记完成（显示绿色打勾样式）。再次点击可撤销。质变仪和探索派遣在点击完成时会弹出时间确认对话框（见对应章节）。

每个号主任务组右侧有 一键完成 按钮，点击后将该号主所有可完成的任务一次性标记为完成（已完成和冷却中的任务不受影响）。页面右上角的 全部标记完成 按钮效果相同，但作用于所有号主。

任务排序规则

每个号主的任务组内，任务按未完成 → 已完成 → 冷却中的顺序从左到右排列。

切换日期与补记

顶部右上角的日期选择器可切换到历史日期进行查看或补记。点击 回到今天 恢复当前游戏日。补记的完成时间以实际点击时刻为准，记录日期归入所选历史日期。

切换到未来日期时进入预览模式：顶部汇总卡片隐藏，显示预览横幅，任务仅供参考，无法对任务进行操作。

自动刷新

查看今日任务时，页面每分钟自动刷新一次（切换到历史或未来日期后不会触发）。质变仪/探索派遣的冷却倒计时归零时，会立即触发刷新，恢复可点击状态，无需手动操作。

任务类型详解

体力 / 狗粮（每日）

每天 04:00 自动重置，每天都会出现在任务列表中。点击标记完成即可，没有额外配置。

树脂记录（可选）：体力卡片右侧有 树脂 按钮，可填写该号主当前的树脂数量，卡片上会显示预计回满时间（按上限 200、每 8 分钟恢复 1 点计算）。此功能完全可选，不填写不影响任何任务逻辑。

周本备注：在「号主与任务」页点击体力任务可勾选周本。若列表中没有需要的周本（如新版本刚出的），可在周本区域下方的输入框中直接输入名称添加。

爱可菲料理（每周）

每周一 04:00 刷新，本周内完成一次即可。

  • 完成当天仍显示为已完成状态（可撤销）
  • 次日起至下周一 04:00 前，该任务不再出现
  • 完成后卡片下方显示下次刷新日期

质变仪（固定间隔 7 天）

质变仪需要记录实际使用时间，以便精确计算 7 天冷却的结束时刻。

  • 点击卡片后弹出 使用时间 确认框，默认为当前时刻，可手动修改
  • 确认后进入 7 天冷却期，卡片变为灰色禁用状态，显示实时倒计时
  • 冷却期内不可点击，冷却结束后页面自动刷新并恢复可点击

若忘记记录使用时间，可在弹出框中手动输入实际使用的时间点，软件从该时间起算 7 天冷却。

壶（固定间隔 3 天）

尘歌壶每 3 个游戏日可进入一次收取奖励。

  • 点击标记完成，3 个游戏日后重新出现
  • 完成后显示下次可收取日期

探索派遣（固定间隔 15 或 20 小时）

探索派遣的冷却时间由号主角色阵容决定，分为 15小时 和 20小时 两种，需要为每个号主单独配置。

  • 配置时长：在「号主与任务」页点击对应号主的探索派遣按钮，在备注设置中选择 20小时 或 15小时（默认 20小时）
  • 点击完成时弹出 派遣时间 确认框，默认当前时刻，可手动修改
  • 确认后进入冷却，卡片变灰显示倒计时，不可点击
  • 冷却结束后自动恢复，始终显示在今日任务列表中（不会移入其他区域）

深境螺旋（每月 16 日刷新）

每月 16 日 04:00 出现，下月 16 日 04:00 前有效。

  • 显示本期截止时间（截止日期 03:59）
  • 完成后当月不再出现，下月 16 日重新刷新

幻想真境剧诗（每月 1 日刷新）

每月 1 日 04:00 出现，下月 1 日 04:00 前有效，逻辑与深境螺旋相同。

危战（版本周期）

危战随每个 42 天大版本开放，版本期间出现在任务列表中。

  • 需要在「设置与备份」中设置 版本日期锚点——填入任意一个已知大版本开始的星期三日期，软件以此为基准每 42 天自动推算后续版本时间窗口
  • 锚点只需设置一次，后续版本自动滚动
  • 版本结束后自动隐藏，下个版本开始时重新出现

限时活动

用于追踪版本内的大活动和小活动任务，可按号主单独开关。

添加活动

在「限时活动」页选择活动分类和时长，填写开始日期（活动名称可选），系统会自动计算截止时间，点击 添加活动 即可加入列表。

为号主启用活动

活动添加后，在「号主与任务」页各号主卡片的「活动」区域会出现对应按钮，点击即可为该号主开启或关闭此任务。不同号主可独立开关同一个活动。

如需为所有号主统一开启，可直接点击「限时活动」页活动条目上的 全部启用 按钮，已启用的号主不受影响。

活动到期

活动截止日期过后，该任务不再出现在今日任务页，但活动条目仍保留在「限时活动」页中供查阅。

剧情任务

用于追踪魔神任务、传说任务和世界任务的通关情况，支持版本限定奖励截止提醒。

任务类型

  • 魔神任务：主线剧情，有版本专属奖励，截止日期为当前大版本结束
  • 传说任务：角色专属剧情，每半个大版本更新一次奖励窗口
  • 世界任务：无时限奖励，不显示截止日期

添加剧情任务

填写任务名称、选择类型和所属号主（从已有号主选择，或填写临时号主名称）。勾选 有版本限定奖励 后，系统根据版本周期自动计算奖励截止时间。

任务排序

未完成任务优先显示，其中有截止时间的按截止日期从近到远排列；完成的任务排在最后，但仍保留在列表中可供查阅。

历史记录

查询指定时间段内的任务完成记录，可按号主筛选。

查询方式

  • 点击 最近一个月 或 最近半年 快捷按钮自动填入日期范围
  • 或手动选择开始和结束日期，修改后立即重新查询
  • 右侧下拉菜单可按号主筛选结果；旁边的任务下拉菜单可只看某一类任务（如体力、深境螺旋）的记录，切换时即时生效无需重新查询

补记说明

在今日任务页切换到历史日期后进行打卡，记录会归入所选历史日期。完成时间戳以实际点击时刻为准。

设置与备份

界面主色调

提供 7 种颜色方案（黑、白、绿、蓝、紫、玫瑰、琥珀），选择后立即生效，下次打开保持当前选择。

角色背景

可上传 PNG、JPEG 或 WebP 格式图片（最大 12 MB）作为界面背景，并调整强度、比例和水平位置。选择黑、白或绿色主题可恢复纯色界面。

危战版本周期

填入任意一个已知原神大版本开始的星期三日期作为 版本日期锚点，软件以此为基准每 42 天计算一次，自动推算危战的开放时间窗口。锚点只需设置一次。

导出备份

将所有号主、任务配置、活动和完成历史导出为 JSON 文件。不包含账号密码（DPAPI 加密绑定本机，无法随文件迁移）。

导入备份

从 JSON 备份文件还原数据。账号密码需在导入后重新填写。

导入后当前所有数据将被完全替换。程序会在导入前自动把当前数据保存为 pre-import-日期时间.json 快照（与数据库同目录，最多保留 5 份），误导入时可用该文件重新导入还原。

版本与更新

程序启动时静默检查 GitHub Releases 是否有新版本，发现更新后在顶部显示横幅提示。可在此处手动检查。确认更新后自动下载并重启，无需手动操作。

清空数据库

删除全部号主、任务和历史记录，恢复初始状态。

此操作不可撤销，执行后无法恢复任何数据。请务必先在「导出备份」处保存一份备份文件。

关闭程序

停止后台服务。已保存的数据不会丢失，下次双击 EXE 即可重新启动。
